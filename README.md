## 📌 Project Overview
**AI Stock Pro** ek real-time, interactive financial forecasting web application hai. Isme humne **LSTM (Long Short-Term Memory) Neural Network** ka use kiya hai jo stock market ke purane trends ko analyze karke aane wale dino ka price predict karta hai.

Is dashboard ko **Glassmorphism** design aur premium colors ke sath banaya gaya hai taaki user ko ek modern FinTech experience mile.

## 🚀 Key Features
* **Live Data:** `yfinance` API ke zariye live market data fetch hota hai.
* **AI Predictions:** Trained LSTM model aage ke 14-60 dino ka forecast karta hai.
* **Interactive Charts:** Professional Candlestick charts aur Moving Averages (MA).
* **Advanced Tables:** `AgGrid` ka use karke raw data ko sort aur filter karne ki suvidha.
* **Modern UI:** Navigation bar aur premium Pink/Dark gradient theme.

## 🛠️ Technology Stack
* **Deep Learning:** PyTorch (LSTM)
* **Data Science:** Pandas, NumPy, Scikit-Learn
* **Frontend:** Streamlit, Streamlit-Option-Menu, AgGrid
* **Visuals:** Plotly Graph Objects
